“幅広い課題に向き合えるエンジニア” を目指して

私は日本電子専門学校でJavaやPHPを中心に学び、授業では仕様書をもとに設計・実装・テスト・発表まで一連の流れを経験してきました。 チーム開発では顧客管理Webアプリを制作し、設計やテストを担当しました。進捗が遅れた場面では役割を調整し、チーム全体で完成させることができました。 私は幅広い分野で活躍できるエンジニアを目指しています。IT開発で培った論理的思考やチームでの問題解決力を基盤に、ロボット分野にも積極的に挑戦し、新しい技術を柔軟に学びながら、現場で信頼されるロボットエンジニアとして成長していきたいと考えております。

日本語学習Webアプリ

PHP (MySQL) と JavaScript を用いて開発した日本語学習アプリです。主な機能として、ユーザー認証（自動登録・ログイン）、学習項目のCRUD操作を実装。特に、Fetch APIによる非同期通信を活用し、ページを再読み込みすることなくリアルタイムでの検索とフィルタリングを可能にした点が特徴です。また、PDOを使用してSQLインジェクション対策を行い、セキュリティ面にも配慮しました。
